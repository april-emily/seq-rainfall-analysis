# South East Queensland Rainfall Analysis
**Author:** April Williams | **Grade:** High Distinction (37/40)

## Overview
A statistical analysis of 110 years of historical weather data to investigate reporting anomalies in regional weather stations.

## Key Techniques
* **R (Tidyverse)** for data cleaning.
* **Chi-Square Testing** to identify "weekend aggregation" bias.
* **ggplot2** for visualisation.
